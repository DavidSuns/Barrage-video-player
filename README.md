# Barrage-video-player
一款强大的视频弹幕播放器，拥有丰富的功能，优美的UI，使用简单方便。

##功能介绍
### 视频功能介绍
* 视频播放，暂停。
* 视频的下一集切换。
* 视频时间显示，进度条拖拽。
* 视频音量调节，静音。
* 视频全屏，取消全屏功能。
* 视频清晰度显示及切换。
* 视频控制的快捷键支持。
* 视频的移动端播放。
* 多段视频源支持。

### 弹幕功能介绍
* 弹幕的播放取消。
* 弹幕发送速度调节。
* 弹幕透明度调节。
* 同屏最大弹幕数调节。
* 弹幕描边宽度调节。
* 发送弹幕。
* 发送弹幕颜色调节。
* 发送弹幕字体大小调节。
* 发送弹幕运动模式调节。
